# Habrahabr

Habrahabr API client.

# Installation

For now, use bundler and install via git. I am currently feeling uncomfortable squatting rubygems entry for habrahabr.

```
gem 'habrahabr', git: 'https://github.com/markiz/habrahabr.git'
```
